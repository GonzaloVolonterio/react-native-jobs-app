# Jobs Tracking App for mobile

### Jobs is tracking app for mobile where you can search for the latest offers work based on the API of RapidApi.

### Built With

```
Expo
React-Native
RapidAPI

```

### Clone repo 🔧

```
https://github.com/GonzaloVolonterio/react-native-jobs-app

```
### Install🔧

```
npm install

npx expo start

```

Or you can Scan QR ,for this you need have the Expo Go app installed.
